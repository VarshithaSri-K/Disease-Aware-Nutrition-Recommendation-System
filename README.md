# Disease-Aware Nutrition Recommendation System

## Overview

A machine learning project that predicts disease categories using nutritional and lifestyle indicators and provides personalized meal recommendations.

## Features

* Data preprocessing and feature encoding
* Random Forest classification
* Disease prediction
* Personalized meal recommendation system
* Feature importance analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Results

* Dataset Size: 1698 records
* Features: 14 input features
* Accuracy: 99.7%

## Future Improvements

* Streamlit Web Application
* Multi-label disease prediction
* Advanced recommendation algorithms
